# Fully Gimbap :leaves:
### commit 규칙
    feat : 새로운 기능에 대한 커밋
    fix : 버그 수정에 대한 커밋
    build : 빌드 관련 파일 수정에 대한 커밋
    chore : 그 외 자잘한 수정에 대한 커밋
    ci : CI관련 설정 수정에 대한 커밋
    docs : 문서 수정에 대한 커밋
    style : 코드 스타일 혹은 포맷 등에 관한 커밋
    refactor :  코드 리팩토링에 대한 커밋
    test : 테스트 코드 수정에 대한 커밋

* 기능 정리 </br></br>
    1. 회원가입 / 로그인
        - jwt 활용
        - 소셜로그인 (네이버, 카카오) : 소셜로그인으로 로그인한 경험이 있다면 알려주기 (참고사이트 : 가비아, 똑닥)
        - 일반로그인 (주민등록수집X, 아이디, 비밀번호, 이름, 핸드폰번호, 이메일) </br></br>
    2. 추천조합
        - 추천조합 클릭하면 그 조합으로 바로 주문 </br></br>
    3. 주문
        - 메뉴 선택(야채류, 돼지고기류,...) -> 토핑선택(택1) -> 야채선택 -> 밥선택 -> 세트/단품선택 -> 사이드 선택 -> 음료 선택 -> 소스선택 </br></br>
    4. 배달
        - 매장 5km 이내의 주소만 가능
        - input 목록은 배달의 민족 참고 </br></br>
    5. 1:1 상담
        - 챗봇시스템 (chatGPT 활용) </br></br>
    6. 공지사항
        - 단순조회 가능 (관리자 로그인 시 수정,삭제,입력 가능)</br></br>
    7. 마이페이지
        - 최근 주문 내역
        - 회원 정보 변경
